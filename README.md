🌐 Enterprise Network Design (Cisco Packet Tracer)
📌 Overview

This project is a group assignment for **Introduction to Networking**.  
It designs a secure, scalable enterprise network for **NetVerse Dynamics**, connecting four branches: Cyberjaya (HQ), Shah Alam, Johor Bahru, and Ipoh.  

The project demonstrates **LAN/WAN topology design, IP addressing with subnetting, static routing configuration, and IoT integration** using Cisco Packet Tracer.  
The `Documentation` file contains detailed branch-by-branch topology, IP schemes, and configuration justifications.

---

🏢 Branch Designs & Topologies
- **Cyberjaya (HQ)** → Star Topology, DNS server, central WAN hub  
- **Ipoh** → Tree Topology, Web server branch  
- **Johor Bahru** → Hybrid (Ring + Star), Mail server branch  
- **Shah Alam** → Bus Topology, File server branch  

---

🧑‍💼 Features
- Logical topology design per branch  
- IP addressing schemes using FLSM/VLSM  
- WAN connectivity with static routing  
- IoT integration (CCTV, sensors, RFID, smart devices)  
- Dedicated server rooms (DNS, Web, File, Mail servers)  
- Justification of design choices for scalability & security

🧠 Key Concepts Demonstrated
- Cisco Packet Tracer simulation  
- LAN/WAN topology design (Star, Tree, Ring, Bus, Hybrid)  
- IP addressing & subnetting (FLSM/VLSM)  
- Static routing configuration  
- Role-based branch design  
- IoT integration in enterprise networks  

---

⚠ Limitations
- Simulated environment only (no real hardware)  
- Static routing only (no dynamic protocols like OSPF/EIGRP)  
- Limited security features (basic firewall rules only)
